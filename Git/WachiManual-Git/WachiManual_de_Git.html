3.- Curso Git - Nuestro primer proyecto - YouTube
https://www.youtube.com/watch?v=vH9pkFf1D7M

Primero ir al directorio donde está mi proyecto, ejem:
cd /home/wachin/github/FFmulticonverter-sin-unoconv/
git init
allí tengo los archivos y debo poner en la terminal:
git init
este comando marca el inicio del proyecto, git emprezará a monitorear todos los cambios que desde allí yo haga. Sólo vamos a usar este comando una sóla vez, y también siempre cuando vamos a iniciar git en nuestro proyecto

git status
git status
este comando me dirá cuantos archivos hay en mi proyecto

git add.- Para agregar uno por uno
git add ejemplo.txt
git add -A.- Para agregarlos todos usaré el comando 
git add -A
Esto es útil para mí porque ejemplo yo quiera pegar allí en esa carpeta de mi proyecto varios archivos y carpetas, con ese somando -A los añadiré todos

git commit -m "Mensaje".- Para guardar los cambios e identificar los cambios en un archivo o varios cambios en varios archivos primero poner:
git status
observaré que la parte donde se ha generado un cambio estará de color rojo, y para guardar esos cambios e identivicar que cambio fue hecho, poner en la terminal:
git commit -m "Mi primer archivo deb en Git"
Guardar nuevos cambios hechos
Repetir el mismo proceso después de hacer un o algunos cambios

Git Log, Mostrar todos los cambios hechos
git log
Nota: el programa gitg es mejor para hacer eso.

git checkout.- Para viajar en el tiempo


git remote.-  Sirve para conectar nuestro repositorio local al que yo tengo en github
